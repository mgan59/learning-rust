# Learning Rust


## Overview

This project repository is a learning environment to teach myself Rust Lang through an interactive approach.  Each folder represents a new coding example and a `GUIDE.md` file that gives instructions on core Rust Lang concepts.  Slowly introducing new topics over each iteration.  The current focus is on building CLI tools as these are a great starting place for learning Rust as there is little need for additional frameworks at the moment.  This will continue to change over time.

Claude-Code is being used as the primary llm-driver.
